# Moon Hub V6 - KAT Script

**Version:** V6 
**Author:** Meowbucks

---

## Overview

**Moon Hub V6** is an advanced script created for the Roblox game **KAT (Knife Ability Test)**. Featuring a robust and easy-to-use **Rayfield GUI**, Moon Hub V6 provides players with powerful tools to dominate the game. With features like **Silent Aim**, **Aimbot**, **ESP**, **Rapid Fire**, and more, this script offers a competitive edge in every match.

---

## Features

- **Silent Aim**: Automatically aim at the nearest player for effortless shots.
- **Rainbow FOV**: A customizable and colorful field of view circle for better target awareness.
- **Aimbot**: Automatically locks your aim onto enemies.
- **ESP**: Displays player locations with custom visuals through walls.
- **Rapid Fire**: Enables ultra-fast shooting with guns.
- **Spam Knife**: Allows you to rapidly throw knives with no cooldown.
- **Bunny Hop**: Continuous jumping to move faster and outmaneuver your enemies.
- **Infinite Jump**: Jump without limits, giving you vertical mobility.
- **Error Handling**: Uses advanced techniques such as `xpcall`, `pcall`, and `task.wait` to ensure smooth execution.

---

## Installation

1. Copy the Moon Hub V6 script.
2. Open **Roblox** and launch **KAT**.
3. Open your preferred script executor.
4. Paste the script into the executor and execute it.
5. Enjoy the enhanced gameplay!

---

## Usage

- Navigate the GUI to enable or disable features like **Aimbot**, **ESP**, and **Rapid Fire**.
- Customize settings for features like **Rainbow FOV** to match your preferences.
- Utilize **Spam Knife** for continuous knife throws, or activate **Infinite Jump** to outmaneuver opponents.

---

## Requirements

- A **Roblox script executor** capable of executing Lua scripts.
- **KAT** (Knife Ability Test) game access.
